# Homebrew Tap for Arc Academy Terminal

Official Homebrew tap for [Arc Academy Terminal](https://github.com/metarobb/arc-academy-terminal) - an interactive TUI for learning shell commands.

## Installation

```bash
brew tap metarobb/arc
brew install arct
```

Or in one command:

```bash
brew install metarobb/arc/arct
```

## Usage

```bash
arct          # Launch interactive TUI
arct --help   # See all options
```
